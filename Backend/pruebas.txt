* **`Authorization: Bearer <tu_token_jwt>`**: Encabezado de autorización con el token JWT obtenido después del inicio de sesión. Reemplaza `<tu_token_jwt>` con el token real.
* **Cuerpo JSON**: Datos necesarios para cada solicitud.

**Recomendaciones:**

* Asegúrate de reemplazar `<tu_token_jwt>` con el token JWT real.
* Verifica que tu backend esté en ejecución y escuchando en el puerto correcto (por defecto, 3000).
* Verifica que mongoDB este levantado.
