-- Registro de Usuario
Datos iniciales
    Nombre + apellido
    correo
    password (se encripta)
Datos finales
    telefono
    fecha nacimiento
    genero
    peso
    estatura
    Objetivo
Nota: Estos datos los puede completar una vez tenga creada su cuenta.

-- Login de Usuario
El usuario se puede logear y tendra Acceso
    Home
    Su perfil donde llenara los datos faltantes
    Registro de Logs ()
    Generacion de Reporte (solo se activara si tiene suficientes datos o pasado 1 mes)

-- Log Out
    Usuario se deslogea


-- Registro de Logs
    Cada log esta relacionado a un usuario
    contiene la fecha
    ejercicio
    serie
    repeteciones
    peso
    notas
Nota: estos Logs son necesarios para crear reportes
